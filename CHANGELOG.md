# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [master]

## [1.1.0] - 2024-12-14

### Changed
- Overlay widget control buttons added
- App icon updated

## [1.0.1] - 2024-12-14

### Fixed
- Fixed a critical bug that caused the standalone executable (.exe) to crash on startup

## [1.0.0] - 2024-12-14

### Added
- Initial release
- Smart game detection with API and Manual modes
- Customizable timer with 5-60 second intervals
- Quick preset buttons (20s, 25s, 30s, 35s)
- Sound alerts with adjustable volume
- Windows popup notifications
- Transparent, draggable in-game overlay
- Session, daily, and all-time statistics tracking
- Multi-language support (English, Turkish, German, Spanish, French)
- System tray integration
- Dark theme UI
